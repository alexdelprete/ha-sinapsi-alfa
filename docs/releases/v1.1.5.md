# Release Notes - v1.1.5

**Release Date:** 2025-12-18

## Overview

**Minor release** - Updates ModbusLink to v1.3.1 with English logging support, fixing mixed Chinese/English log messages.

## Improvements

### English-Only Logging

ModbusLink v1.3.1 introduces configurable logging language. The integration now explicitly configures English logging to fix the mixed Chinese/English messages that appeared in error logs.

**Before:** `事务ID不匹配 | Transaction ID mismatch: 期望 | Expected 5, 收到 | Received 1`
**After:** `Transaction ID mismatch: Expected 5, Received 1`

### Updated ModbusLink Dependency

Bumped the modbuslink requirement from `>=1.2.0` to `>=1.3.1` to access the new language configuration feature.

## Code Changes

### api.py

Added imports and configuration:

```python
from modbuslink import Language, ModbusLogger

# Configure ModbusLink to use English logging
ModbusLogger.setup_logging(language=Language.EN)
```

## Files Changed

| File | Changes |
|------|---------|
| `api.py` | Added `Language` and `ModbusLogger` imports, configured English logging |
| `manifest.json` | Updated modbuslink requirement to `>=1.3.1` |
