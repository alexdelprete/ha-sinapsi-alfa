# v1.2.0

**Release Date:** 2025-12-29

**Type:** Stable Release - Gold Tier Quality Scale

## Highlights

This release achieves **Gold tier** compliance with the
[Home Assistant Integration Quality Scale](https://developers.home-assistant.io/docs/core/integration-quality-scale/),
bringing enterprise-grade features to the integration.

## What's New

### Gold Tier Quality Scale Features

#### Diagnostics Support

- **Download diagnostics** from Settings > Devices & Services > Alfa by Sinapsi > 3-dot menu > Download Diagnostics
- Provides debug information including:
  - Integration configuration (host/port redacted for privacy)
  - Current coordinator state
  - Last update timestamp
  - All sensor data

#### Repair Issues

- **Automatic repair issue creation** after 3 consecutive connection failures
- Repair issue appears in Settings > System > Repairs
- Provides troubleshooting guidance for connection problems
- Issue auto-resolves when connection is restored

#### Translations (10 Languages)

- **Full translation support**: German (de), English (en), Spanish (es), Estonian (et),
  Finnish (fi), French (fr), Italian (it), Norwegian Bokmal (nb), Portuguese (pt), Swedish (sv)
- All sensor names, config flow labels, and error messages translated
- Exception messages use Home Assistant's translation system

#### Improved Terminology

- Changed sensor terminology from "Drawn/Fed" to "Imported/Exported"
- More intuitive for energy monitoring context
- Applied consistently across all 10 language translations

#### Icon Translations

- Custom icons defined in `icons.json` for all 24 sensors
- Proper icon display in entity lists and dashboards

#### Entity Disabled by Default

- F1-F6 time band sensors (12 sensors) now disabled by default
- Users can enable them in Settings > Devices & Services > Alfa by Sinapsi > Entities
- Reduces clutter for users who don't need time band tracking

### Code Quality Improvements

- **Full type annotations** for mypy compliance (Platinum strict-typing rule)
- **Comprehensive test suite** with 95% coverage threshold
- **pyproject.toml consolidation** for modern Python packaging
- **GitHub Actions CI** with automated testing
