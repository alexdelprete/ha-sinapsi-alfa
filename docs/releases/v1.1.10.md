# Release Notes - v1.1.10

**Release Date:** TBD

## Overview

**Patch release** - Updates ModbusLink to 1.3.2 for English-only error messages.

## Improvements

### ModbusLink 1.3.2 Update

- **English-only errors** - Exception messages are now fully in English (previously mixed Chinese/English)
- **Simpler language API** - Uses new `set_language()` function that affects both logs AND errors
- **Cleaner imports** - `Language` and `set_language` now imported directly from `modbuslink` package

Thanks to the ModbusLink developer ([@Miraitowa-la](https://github.com/Miraitowa-la)) for the quick fix!

## Files Changed

- **`api.py`** - Updated import and language configuration
- **`manifest.json`** - Updated ModbusLink requirement to `>=1.3.2`

## Requirements

- Home Assistant 2025.10.0 or newer
- Python 3.13.2 or newer
- ModbusLink 1.3.2 or newer

## Upgrade Notes

- No configuration changes required
- Error messages will now be fully in English
