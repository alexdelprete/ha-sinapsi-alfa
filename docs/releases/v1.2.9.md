## What's Changed

### New Features

- **ModbusLink version in startup log** - The startup message now includes the installed modbuslink
  version for easier debugging of dependency issues.

**Before:**

```text
Sinapsi Alfa - Version 1.2.9
```

**After:**

```text
Sinapsi Alfa - Version 1.2.9 (modbuslink 1.4.1)
```
