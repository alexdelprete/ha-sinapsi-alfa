# Release Notes - v1.1.7

**Release Date:** 2025-12-19

## Overview

**Patch release** - Fixes ModbusLink import path for Language and ModbusLogger classes.

## Bug Fixes

### Fixed ModbusLink Import Path

The `Language` and `ModbusLogger` classes are not exported from the main `modbuslink` package - they must be imported from the `modbuslink.utils.logging` submodule.

**Before (v1.1.5/v1.1.6 - broken):**

```python
from modbuslink import Language, ModbusLogger  # ImportError
```

**After (v1.1.7 - fixed):**

```python
from modbuslink.utils.logging import Language, ModbusLogger  # Works
```
